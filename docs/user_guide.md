# StylusCore User Guide

## Overview

StylusCore is a highly customizable note-taking application designed for graphics tablet users.

## Getting Started

### First Launch
- Device calibration wizard will help you set up your tablet
- Choose your preferred language and theme

### Basic Controls
- **Keyboard + Mouse Mode**: Type text, click to select
- **Graphics Tablet Mode**: Draw with pen, pressure-sensitive

## Radial Menu
- Customizable circular menu
- Opens at cursor position
- Hold or click to select items
